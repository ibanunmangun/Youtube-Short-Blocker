# YouTube Shorts Block Extension

## Objective
The goal of this project is to create a Chrome extension that automatically hides or blocks "Shorts" content from the YouTube interface (including the sidebar, homepage, and search results).

## Deliverables
- A functional Chrome Extension (`src/` folder).
- User guide for installation and usage (`docs/Walkthrough.md`).
- Technical documentation (`docs_plan/Planning.md`).
